## Features

- Accent colors drawn from official TfL tube line colours
- Deep navy backgrounds inspired by Underground tunnels
- WCAG AA contrast validated — never ships unreadable text
- Semantic highlighting with fine-grained token colors
- Full editor UI theming (sidebar, tabs, status bar, terminal, git decorations)

## Also available for

Neovim, Zed, Kitty, Alacritty, WezTerm, Ghostty, iTerm2, foot, Windows Terminal, tmux, fzf, bat, delta, lazygit, and OpenCode.
